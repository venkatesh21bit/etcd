📋 Raft Log Replication

All writes flow through the Raft leader via AppendEntries RPCs. Once a majority of nodes acknowledge, the entry is committed and applied to every node's state machine — guaranteeing strong consistency.

Waiting for election…

💓 Raft Heartbeat Monitor

The Raft leader broadcasts periodic empty AppendEntries messages as heartbeats to keep followers from starting a new election. If heartbeats stop arriving, a follower's election timeout fires and it becomes a Candidate.

Waiting for election…

👑 Bully Algorithm Election

The Bully Algorithm elects the node with the highest priority as leader. Lower-priority nodes send ELECTION messages to higher-priority nodes. If a higher node responds with OK, the sender steps down. The surviving highest-priority node broadcasts COORDINATOR to all.
